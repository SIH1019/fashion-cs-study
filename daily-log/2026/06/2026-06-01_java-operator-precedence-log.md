## 4. 연산자란?

- 연산자는 계산을 할 때 사용하는 기호이다.
- Java에서는 숫자 계산, 문자열 연결, 값 처리 등에 연산자를 사용한다.
- 변수에 저장된 값을 이용해서 새로운 결과를 만들 수 있다.

예시:

```java
int a = 10;
int b = 3;

System.out.println(a + b);
```

- `+`
  - 덧셈 연산자

- `a + b`
  - 변수 `a`와 `b`에 저장된 값을 더하는 연산식

---

## 5. 산술 연산자

- 산술 연산자는 숫자 계산에 사용하는 연산자이다.
- 덧셈, 뺄셈, 곱셈, 나눗셈, 나머지 계산을 할 수 있다.

정리:

```text
+  → 더하기
-  → 빼기
*  → 곱하기
/  → 나누기
%  → 나머지
```

예시:

```java
int a = 10;
int b = 3;

System.out.println(a + b);
System.out.println(a - b);
System.out.println(a * b);
System.out.println(a / b);
System.out.println(a % b);
```

실행 결과:

```text
13
7
30
3
1
```

---

## 6. 덧셈 연산자

```java
int a = 10;
int b = 3;

System.out.println(a + b);
```

- `+`는 두 값을 더할 때 사용한다.
- `10 + 3`의 결과는 `13`이다.

정리:

```text
10 + 3 = 13
```

---

## 7. 뺄셈 연산자

```java
int a = 10;
int b = 3;

System.out.println(a - b);
```

- `-`는 왼쪽 값에서 오른쪽 값을 뺄 때 사용한다.
- `10 - 3`의 결과는 `7`이다.

정리:

```text
10 - 3 = 7
```

---

## 8. 곱셈 연산자

```java
int a = 10;
int b = 3;

System.out.println(a * b);
```

- `*`는 두 값을 곱할 때 사용한다.
- `10 * 3`의 결과는 `30`이다.

정리:

```text
10 * 3 = 30
```

---

## 9. 나눗셈 연산자

```java
int a = 10;
int b = 3;

System.out.println(a / b);
```

- `/`는 왼쪽 값을 오른쪽 값으로 나눌 때 사용한다.
- `10 / 3`의 결과는 `3`이다.
- `int`끼리 나누면 결과도 정수로 나온다.
- 소수점 아래 값은 버려진다.

정리:

```text
10 / 3 = 3
```

- 수학적으로는 `3.333...`이지만, Java에서 정수끼리 나누면 정수 부분만 남는다.

---

## 10. 나머지 연산자

```java
int a = 10;
int b = 3;

System.out.println(a % b);
```

- `%`는 나머지를 구하는 연산자이다.
- `10 % 3`은 `10`을 `3`으로 나누고 남은 값을 구한다.
- 결과는 `1`이다.

정리:

```text
10 / 3 = 몫 3, 나머지 1
10 % 3 = 1
```

- 나머지 연산자는 나중에 짝수/홀수 판단, 배수 판단에서 사용할 수 있다.

예시:

```java
int number = 10;

System.out.println(number % 2);
```

- 결과가 `0`이면 짝수로 판단할 수 있다.
- 결과가 `1`이면 홀수로 판단할 수 있다.

---

## 11. 문자열 더하기

- `+` 연산자는 숫자끼리 사용하면 덧셈을 한다.
- 문자열과 함께 사용하면 문자열을 연결한다.

예시:

```java
String result = "hello" + " java";

System.out.println(result);
```

실행 결과:

```text
hello java
```

- `"hello"`와 `" java"`가 이어져서 `"hello java"`가 된다.

---

## 12. 문자열과 숫자 더하기

```java
System.out.println("result = " + 10);
```

실행 결과:

```text
result = 10
```

- 문자열과 숫자를 `+`로 연결하면 숫자가 문자열처럼 이어 붙는다.
- 문자열이 하나라도 포함되면 `+`는 문자열 연결 역할을 한다.

---

## 13. 문자열 더하기 실행 순서

### 13-1. 문자열이 먼저 나오는 경우

```java
System.out.println("result = " + 10 + 20);
```

실행 결과:

```text
result = 1020
```

- `"result = " + 10`이 먼저 문자열 연결로 처리된다.
- 그 결과 `"result = 10"`이 된다.
- 이후 `20`도 문자열처럼 이어 붙는다.
- 최종 결과는 `"result = 1020"`이다.

---

### 13-2. 숫자가 먼저 나오는 경우

```java
System.out.println(10 + 20 + " result");
```

실행 결과:

```text
30 result
```

- 앞의 `10 + 20`이 먼저 계산된다.
- 계산 결과는 `30`이다.
- 그 다음 `" result"`와 문자열로 연결된다.
- 최종 결과는 `"30 result"`이다.

## 14. 연산자 우선순위

- 연산자 우선순위는 여러 연산자가 함께 있을 때 어떤 연산을 먼저 할지 정하는 규칙이다.
- 수학에서 곱셈을 덧셈보다 먼저 계산하는 것과 비슷하다.
- Java에서도 연산자마다 계산 순서가 있다.

예시:

```java
int result = 2 + 3 * 4;

System.out.println(result);
```

실행 결과:

```text
14
```

- `3 * 4`가 먼저 계산된다.
- 그 결과는 `12`이다.
- 그 다음 `2 + 12`가 계산된다.
- 최종 결과는 `14`이다.

---

## 15. 괄호로 우선순위 변경하기

```java
int result = (2 + 3) * 4;

System.out.println(result);
```

실행 결과:

```text
20
```

- 괄호 안의 `2 + 3`이 먼저 계산된다.
- 그 결과는 `5`이다.
- 그 다음 `5 * 4`가 계산된다.
- 최종 결과는 `20`이다.

---

## 16. 오늘 배운 연산자 우선순위

- 괄호가 가장 먼저 계산된다.
- 곱셈, 나눗셈, 나머지가 덧셈, 뺄셈보다 먼저 계산된다.
- 덧셈과 문자열 연결은 왼쪽에서 오른쪽 순서로 처리된다.
- 우선순위가 헷갈리면 괄호를 사용하면 된다.

정리:

```text
1. 괄호 ()
2. 곱셈 / 나눗셈 / 나머지 *, /, %
3. 덧셈 / 뺄셈 +, -
4. 문자열 연결 +
```

---

## 17. 우선순위보다 괄호를 우선 사용하기

- 연산자 우선순위를 모두 외우려고 하지 않아도 된다.
- 헷갈리는 식은 괄호를 사용하면 된다.
- 괄호를 사용하면 계산 순서가 명확해진다.
- 코드를 읽는 사람도 이해하기 쉬워진다.

예시:

```java
int result = (2 + 3) * 4;
```

- 괄호를 사용해서 먼저 계산할 부분을 명확히 표시했다.

## 18. 오늘 작성한 예제 코드

```java
package operator;

    }
}
```

```text

1
result = 1020

14
20
```

## 19. 오늘 새로 알게 된 점

- 연산자는 계산이나 문자열 연결에 사용된다.
- 산술 연산자에는 `+`, `-`, `*`, `/`, `%`가 있다.
- `int`끼리 나누면 결과도 정수로 나온다.
- `/`는 나눗셈 결과의 몫을 구할 때 사용된다.
- `%`는 나머지를 구할 때 사용된다.
- `+`는 숫자끼리 사용하면 덧셈이다.
- `+`는 문자열과 함께 사용하면 문자열 연결이다.
- 문자열과 숫자를 더하면 숫자가 문자열처럼 이어 붙는다.
- 연산자에는 계산 순서가 있다.
- 곱셈, 나눗셈, 나머지는 덧셈, 뺄셈보다 먼저 계산된다.
- 괄호를 사용하면 계산 순서를 직접 정할 수 있다.

---

## 20. 헷갈린 부분

- `/`와 `%`의 차이가 헷갈릴 수 있다.
- `int / int` 결과가 소수점 없이 정수로 나오는 점이 헷갈릴 수 있다.
- 문자열과 숫자를 `+`로 연결할 때 계산이 아니라 문자열 연결이 되는 경우가 헷갈렸다.
- `"result = " + 10 + 20`과 `10 + 20 + " result"`의 결과가 다른 점이 헷갈릴 수 있다.
- 연산자 우선순위를 모두 외우기보다는 괄호를 사용하는 것이 더 명확하다고 느꼈다.

## 21. 내 프로젝트와 연결한 부분

- 패션 추천 시스템에서 점수를 계산할 때 산술 연산자를 사용할 수 있다.
- 색상 점수, 핏 점수, 기장 점수를 더해서 전체 점수를 계산할 수 있다.
- 문자열 더하기를 사용해서 분석 결과 문장을 만들 수 있다.
- 점수 계산식이 복잡해지면 괄호를 사용해서 계산 순서를 명확히 해야 한다.

예시:

```java
int colorScore = 82;
int fitScore = 78;
int lengthScore = 75;

int totalScore = colorScore + fitScore + lengthScore;

System.out.println("전체 점수 = " + totalScore);
```

정리:

```text
colorScore  → 색상 점수
fitScore    → 핏 점수
lengthScore → 기장 점수
totalScore  → 전체 점수
```
