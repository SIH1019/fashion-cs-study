## 4. 오늘 이해한 핵심 내용

### 4-1. 변수의 필요성

- 같은 값을 코드에 여러 번 직접 쓰면 나중에 수정이 어려움
- 값을 한 곳에 저장하고 여러 번 사용하기 위해 변수가 필요함
- 변수는 데이터를 담아두는 저장 공간처럼 이해할 수 있음

---

### 4-2. 변수 선언

```java
int a;
```

- `int`는 정수를 저장하는 타입
- `a`는 변수 이름
- `int a;`는 정수를 저장할 수 있는 공간을 만드는 코드
- 변수를 만드는 것을 변수 선언이라고 함

---

### 4-3. 변수 초기화

```java
a = 10;
```

- 선언한 변수에 처음으로 값을 저장하는 것을 초기화라고 함
- Java에서 `=`는 오른쪽 값을 왼쪽 변수에 저장한다는 뜻
- 수학에서 말하는 “같다”와는 다름

---

### 4-4. 변수 값 읽기

```java
System.out.println(a);
```

- 변수 이름을 사용하면 변수에 저장된 값을 읽을 수 있음
- 변수 값을 읽어도 값이 사라지지 않음
- 같은 변수는 여러 번 사용할 수 있음

---

### 4-5. 변수 값 변경

```java
a = 10;
System.out.println(a);

a = 50;
System.out.println(a);
```

- 변수 값은 나중에 변경할 수 있음
- `a = 50;`을 실행하면 기존 값 `10`은 사라지고 `50`으로 바뀜
- 이후 `a`를 출력하면 변경된 값이 출력됨

---

### 4-6. 변수는 초기화해야 함

```java
int a;
System.out.println(a);
```

- 위 코드는 오류가 발생함
- 이유는 변수 `a`에 값이 들어있지 않기 때문
- Java는 초기화하지 않은 지역 변수를 사용할 수 없게 막음

오류 예시:

```text
java: variable a might not have been initialized
```

---

## 5. 오늘 새로 알게 된 점

- 변수는 단순히 값을 담는 문법이 아니라, 코드를 관리하기 쉽게 만드는 기본 도구임
- `=`는 같다는 뜻이 아니라 저장한다는 뜻임
- 변수 값을 변경하면 기존 값은 유지되지 않고 새 값으로 바뀜
- 초기화하지 않은 변수는 사용할 수 없음
- Java는 이런 문제를 컴파일 단계에서 미리 알려줌
- 컴파일 에러는 짜증나는 오류가 아니라, 실행 전에 문제를 알려주는 장치라고 볼 수 있음

---

## 6. 내 프로젝트와 연결한 부분

패션 추천 시스템에서도 변수는 가장 기본적인 재료가 됨.

예시:

```java
String topColor = "navy";
String bottomColor = "black";
String shoesColor = "white";

int colorScore = 82;
int fitScore = 78;
int totalScore = colorScore + fitScore;
```

- `topColor`
  - 상의 색상 저장

- `bottomColor`
  - 하의 색상 저장

- `shoesColor`
  - 신발 색상 저장

- `colorScore`
  - 색상 조합 점수 저장

- `fitScore`
  - 핏 점수 저장

- `totalScore`
  - 전체 코디 점수 저장

---

## 7. 프로젝트 관점에서 느낀 점

- 패션 추천 알고리즘을 만들려면 먼저 옷 정보를 변수로 저장해야 함
- 색상, 핏, 기장, 상황 같은 정보도 모두 변수로 표현할 수 있음
- 지금은 값을 직접 넣는 수준이지만, 나중에는 사용자가 입력한 값을 변수에 저장하게 될 것 같음
- 조건문을 배우면 변수 값에 따라 다른 판단을 내릴 수 있음

예시 방향:

```java
if (topColor.equals("black") && bottomColor.equals("black")) {
    System.out.println("전체적으로 어두운 조합입니다.");
}
```

- 앞으로 조건문을 배우면 패션 도메인 지식을 코드로 판단하는 첫 알고리즘을 만들 수 있을 것 같음

## 9. 오늘의 핵심 정리

- 변수는 값을 저장하는 공간
- 변수는 데이터를 담는 그릇처럼 이해할 수 있음
- `int a;`는 변수 선언
- `a = 10;`은 변수에 값을 저장하는 코드
- 처음 값을 저장하는 것을 초기화라고 함
- 변수 값은 읽을 수 있고, 여러 번 읽어도 사라지지 않음
- 변수 값은 나중에 변경할 수 있음
- 초기화하지 않은 변수는 사용할 수 없음
- 패션 추천 시스템에서도 옷 정보와 점수를 저장하기 위해 변수가 필요함
