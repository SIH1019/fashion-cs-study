Fashion CS Study

컴퓨터공학 기초를 다시 다지고, 패션 추천 시스템을 만들기 위한 학습 기록 저장소

1. 저장소 목적

컴퓨터공학 기초를 다시 공부하고, 그 내용을 실제 프로젝트에 적용하기 위해 만듦

최종적으로는 색상, 핏, 기장, 상황 등을 분석해서 옷 조합을 추천하고 설명해주는 패션 추천 시스템을 만드는 것이 목표

복학 전까지는 완성형 앱을 바로 만드는 것보다, Java 기본기와 객체지향을 먼저 다지고, 작은 콘솔 프로그램으로 추천 로직을 구현해보는 것을 1차 목표

3. 프로젝트 방향

내가 만들고 싶은 시스템은 단순히 옷을 추천하는 앱이 아님

옷 조합을 다음 요소로 나누어 분석하고, 왜 어울리는지 또는 왜 어색한지를 설명하는 논리 기반 패션 추천 시스템을 목표

- 색상
- 명도
- 채도
- 핏
- 기장
- 상황
- 분위기
- 개선 방향

예를 들어 네이비 상의와 블랙 하의를 선택했을 때 단순히 “추천합니다”라고 출력하는 것이 아니라, 다음과 같이 설명하는 시스템 목표

```text
네이비와 블랙은 차분하고 시크한 조합입니다.
다만 상하의가 모두 어두운 계열이라 전체적으로 답답해 보일 수 있습니다.
화이트 신발이나 밝은 이너를 추가하면 명도 대비가 생겨 균형이 좋아집니다.
```
